# 🏛️ Colisseum - Landing Page Premium Dark Mode

## Descripción
Landing page premium en modo oscuro para **Colisseum**, un centro de gaming y eSports inspirado en la grandeza del Coliseo Romano. Diseñada con estética "Dark Luxury Gaming".

## 🚀 Cómo Usar
1. Abre `index.html` en tu navegador o servidor local
2. Si usas XAMPP: accede a `http://localhost/Colliseum`

## ✏️ Personalización
- Colores: edita las variables CSS en `:root` en `styles.css`
- Contenido: modifica textos directamente en `index.html`
- Imágenes: reemplaza archivos en `/images/`
- Links de redes sociales: actualiza los `href` en el footer y CTA
